Notes for the Printed Edition (paper version)

A printed version of the manual is enclosed with each product. For environmental reasons, the document was reduced in size and printed on DIN A5.

Should you experience difficulties reading the font size of the scaled-down printed version, you can print and use the PDF version in DIN A4 format 1:1. You can find the PDF version on the CD-ROMDVD accompanying the product and on the website.
