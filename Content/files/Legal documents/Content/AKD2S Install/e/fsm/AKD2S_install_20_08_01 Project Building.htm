Project building (online)

Offline project building is possible. In Step 4 you can select if you want to work online or offline.

  • Start on the computer.


  • Create a new project.

  • Add a device to the project.


  • Select the device.


  • Select Safety Functionality


  • Set Password (example for full configuration)


    If you left the project and want to re-open it later, you need the password.


  • Save the project (Example: Demo-AKD2G)

  • Select Safety.

  • Select the Safety Group (rename if required) and add the device to the group.

  • Select the Parameters Editor for parameterizing the safety functions.
