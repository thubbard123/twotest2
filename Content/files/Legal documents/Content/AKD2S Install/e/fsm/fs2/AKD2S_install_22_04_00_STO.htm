STO (Safe Torque Off)

Safe Torque Off description for drive option Functional Safety 2 or 3.

Description

STO turns off the drive output stage that powers the motor. STO function corresponds to an uncontrolled braking according to IEC 60204-1, stop category 0.

Important Notes

WARNING

No Brake Power!

Serious injury could result when a suspended load is not properly blocked. The drive cannot hold a vertical load when STO is active.

  • Add a safe mechanical blocking (for instance, a motor holding brake).
  • Configure Safe Brake Control SBC (➜ # 1).

It is not possible to perform a controlled brake according to IEC 60204-1 category 1, if STO function is active as standalone function. If controlled braking category 1 is necessary, you should use the SS1-t function (➜ # 1) SS1-t/r function (➜ # 1).

Function Input/Output variables

Inputs

  • STO Activate can be triggered from
  • FSoE, or
  • SS1, or
  • a fault reaction, or
  • safe digital inputs (must be mapped).
  • STO Restart Acknowledge is used for manual restart. The signal can be triggered from
  • FSoE, or
  • safe digital inputs (must be mapped).

Outputs

  • STO Active, logical status of the STO function
  • STO Restart Request is used for manual restart. The signal can be evaluated by using FSoE or safe digital outputs (must be mapped).

 

Activation

The safety function can be mapped with or by using FSoE to the safe digital inputs or other functions.

Activation with configurable time after SS1-t or as SS1 fault reaction

Refer to SS1-t function (➜ # 1) SS1-t/r function (➜ # 1).

Activation by FSoE

For drive profile overview refer to (➜ # 1). For setup and handling refer to the EtherCAT Manual.

Activation by safe digital inputs

Technical data of the safe inputs (➜ # 1) If activation by safe digital inputs is required, then the digital STO inputs can be connected

  • to transistor outputs of a safety control, or
  • to relay outputs of a safety control, or
  • to an emergency stop switch.

If an STO input goes open-circuit or 0 V, then power supply to the motor stops within 6 ms. The motor will lose all torque and coast to a stop.

Review the enclosure and wiring instructions (➜ # 1).

Example: STO, dual channel, 2 axes, SIL3 PLe, Safety Control, electronic outputs, automatic restart. DISTINCTPULSE must be deactivated.

 

Example: STO, dual channel, 2 axes, SIL3 PLe, Safety Control, relay contacts, automatic restart. DISTINCTPULSE must be deactivated.

Safety level must be evaluated referring to B10d value of the relay contacts.

 

Example: STO, dual channel, 2 axes, SIL3 PLe, Emergency Stop, automatic restart. DISTINCTPULSE must be deactivated.

Safety level must be evaluated referring to B10d value of the emergency stop contacts.

 

Example: STO dual channel input, one axis, SIL3 PLe, Emergency Stop, manual restart. DISTINCTPULSE must be deactivated.

Safety level must be evaluated referring to B10d value of the emergency stop contacts.

Restart

Restart can be set as "automatic restart" or "manual restart" (STO Restart Mode, AXIS#.SAFEPARAM.STO.RESTARTACKMODE).

  • With manual restart a restart request signal will be send to the safety control, when STO is deactivated. The axis waits for the restart acknowledge and axis enable to restart.
  • With automatic restart the axis restarts when STO is deactivated and axis is enabled.

Setup in Workbench

Select the drive and open the Safety Parameterization view (➜ # 1).

Choose the axis (1), select category (2), safety function (3). The instance number (4) is not applicable for STO.

The safety parameters are displayed and can be configured.

If you want to have one dual input (e.g. SIN1/SIN2) activate STO for both axis 1 and axis 2, then you must set the activation parameter for both axes to the identical input (e.g. Safe Dual Input 1).

Fault Reaction/ Failure Messages

If dual channel mode is selected, the drive supervises if the two inputs are in a different state for longer than 100 ms. If the two inputs are in different state for more than 100 ms, an I/O failure activates SS1_1 and then the axis torque is switched off by STO.
An I/O failure warning is messaged when the drive is disabled or a fault when the drive is enabled and AXIS#.SAFE.STO.REPORTFAULT is set to 1.

Safety Diagnostic for safe I/Os view in

The view "Safety Diagnostic - Safe I/O" shows the current status on the safe inputs and the logical status of the STO function for every axis.
