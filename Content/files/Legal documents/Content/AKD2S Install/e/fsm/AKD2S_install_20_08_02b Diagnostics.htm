Diagnostics

For help with finetuning, debugging and verification of your safety project, Diagnostics offers live information about internal safety related values.

Displays the current configuration and values of the safe I/Os. It includes the dual configuration information for inputs and outputs and the source of the output signal.

For details refer to the online help.

Displays the current safe brake configuration with the test status including the time remaining to the next test. The safe brake test can also be started and monitored from within that view.

For details refer to the online help.

Information such as function states, safe velocity, safe position and safe acceleration can either be monitored directly or added to the Watch Panel for permanent surveillance or added to the internal oscilloscope. Maximum two values can be observed simultaneously.

For details refer to the online help.

Displays the log stored in the SMM. All actions related to SafeID and parameters validation are logged with username and date whenever possible.

For details refer to the online help.
