Workbench Safety Function Setup

In general, three steps need to be done to activate a safety configuration within the SMM:

  • Configuration of the safety parameters.
  • Download the safety parameters (➜ # 1) to the drive through:
    - Workbench (Telnet): persistent, or
    - EtherCAT (FoE): persistent, or
    - EtherCAT (FSoE) : non persistent.
  • Validation of the safety parametrization by the user.

The safety parameter configuration can be easily done through . The safety parameter configuration, download and validation are completely integrated.

performs a consistency check which helps to identify contrary configurations. If the configuration passes the consistency check, creates a safety parameter file. This file can be download to the drive through or can be exported. Once the safety parameters are configured, downloaded and validated through , the safety configuration will be persistent and no new configuration will be necessary after power cycle.

Preconditions

  • Service port X20 (➜ # 1) is connected by using Ethernet cable to the computer or to a network switch.
  • Auxiliary voltage is connected to X10 and switched on.
  • setup software is installed.
