AXIS#.SAFE.FSOE.APPLPARAMLENGTH

Description

Indicates the size of the FSoE application parameters. If using safety parameters, this is 0 and if using application parameters over FSoE it is 4 bytes.

Context

For complete FSoE information refer to the EtherCAT Manual.
