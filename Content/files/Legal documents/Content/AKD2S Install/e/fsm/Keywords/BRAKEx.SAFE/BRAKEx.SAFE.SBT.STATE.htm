BRAKE#.SAFE.SBT.STATE

Description

Read the state of the Safe Brake Test (SBT) on the specified brake. The return value is one of the following:

Value

Description

0

The brake is not tested.

1

The brake has been successfully tested.

2

The brake failed to test. It may be broken.

3

The brake is applying.

4

The brake is tested.

5

The brake is releasing.

Code

--> BRAKE1.SAFE.SBT.STATE
0

Context

For complete SBT information refer to "SBT (Safe Brake Test)" (➜ # 1).
