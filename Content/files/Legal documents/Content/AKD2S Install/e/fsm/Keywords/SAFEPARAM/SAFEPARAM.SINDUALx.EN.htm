SAFEPARAM.SINDUAL#.EN

Description

Enables dual safe inputs for SIN (Safe INput)1-4.


  • The corresponding single safety inputs are not allowed to be used when being used in a dual configuration. Safety functions must be configured with the dual input.

  • When enabled, paired inputs must be transitioned together within a certain period or an IO failure warning (W9011 ) will occur putting the drive in a safe state.

Context

For more information, see "Safe digital inputs" (➜ # 1)
