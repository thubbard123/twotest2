AXIS#.SAFEPARAM.SBT.MAXTIMEINTERVAL

Description

Indicates the time in minutes that Safe Brake Test (SBT) must be executed before a warning becomes active that indicates SBT should be run and the Safe Brake Acknowledge (SBA) signal becomes low.


  • If the SBT period has elapsed the drive will permit enabling and movement. It is up to the safety system design to prevent motion if required.

 

Context

For complete SBT information refer to SBT (Safe Brake Test).
