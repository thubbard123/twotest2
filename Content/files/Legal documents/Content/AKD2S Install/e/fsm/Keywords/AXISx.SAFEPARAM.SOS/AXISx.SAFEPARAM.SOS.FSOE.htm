AXIS#.SAFEPARAM.SOS.FSOE

Description

Indicates if the Safe Operating Stop (SOS) function can be activated through FSoE. This parameter can only set to “Used” when FSoE is activated (SAFEPARAM.SAFEFIELDBUS.ACTIVATION is set to 1 – “Activated”).

Value Description

0

Not used

1

Used

 

Context

For complete SOS information refer to SOS (Safe Operating Stop).
