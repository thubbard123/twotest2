AXIS#.SAFEPARAM.UNIT.ACCLABEL

Description

User unit label for acceleration values returned by the safety system. This parameter is only required for the parameter visualization in or any other GUI. It is not used by the SMM software.

This is only applicable when a safe encoder has been defined in the safety parameters.
