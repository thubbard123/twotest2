SAFE.MONITOR#.AXIS

Description

Defines the axis of the monitored SMM variable returned by SAFE.MONITOR#.DATA. Two variables can be configured at the same time using SAFE.MONITOR1.AXIS and SAFE.MONITOR2.AXIS.

Code

--> SAFE.MONITOR1.AXIS 1
