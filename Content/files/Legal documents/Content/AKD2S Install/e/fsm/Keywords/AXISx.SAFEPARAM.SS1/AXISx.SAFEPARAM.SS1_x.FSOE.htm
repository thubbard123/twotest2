AXIS#.SAFEPARAM.SS1_#.FSOE

Description

Indicates if the Safe Stop 1 (SS1) function instance can be activated through FSoE. This parameter can only be set to “Used” when FSoE is activated (SAFEPARAM.SAFEFIELDBUS.ACTIVATION is set to 1 – “Activated”).

Value Description

0

Not used

1

Used

 

Context

For complete SS1 information refer to SS1-t (Safe Stop 1)SS1-t/r (Safe Stop 1).
