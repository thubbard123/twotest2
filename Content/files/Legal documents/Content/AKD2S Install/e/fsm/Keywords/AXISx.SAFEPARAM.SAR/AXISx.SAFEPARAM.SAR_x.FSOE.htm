AXIS#.SAFEPARAM.SAR_#.FSOE

Description

Indicates if the Safe Acceleration Range (SAR) safety function can be activated through FSoE. This parameter is only set to “Used” when FSoE is activated (SAFEPARAM.SAFEFIELDBUS.ACTIVATION is set to 1 – “Activated”). When activated, the safety system forbids the axis to move in the negative direction.

Value

Description

0

Not used

1

Used
