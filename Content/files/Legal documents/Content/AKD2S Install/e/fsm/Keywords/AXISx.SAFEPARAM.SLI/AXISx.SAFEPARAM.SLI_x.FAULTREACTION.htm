AXIS#.SAFEPARAM.SLI_#.FAULTREACTION

Description

Indicates the safety function activated when the Safely Limited Increment (SLI) safety function is faulted. The parameter uses one of the following values:

Value Description

0

STO

1

SS1 Instance 1

2

SS2 Instance 1
