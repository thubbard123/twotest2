AXIS#.SAFE.STO.ACTIVE

Description

Indicates the Safe Torque Off (STO) state of the axis. The STO status is an AND link of A AXIS#.SAFE.STO.A and B AXIS#.SAFE.STO.B signals.

Value

Description

1

STO active (inputs +0V)

0

STO inactive (inputs 24V)

Context

For complete STO information refer to STO (Safe Torque Off).
