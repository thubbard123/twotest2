AXIS#.SAFEPARAM.ABSOLUTEPOSEN

Description

Indicates whether the absolute position support is enabled or not. If in use, the absolute position will be valid if the current value read in the encoder by the SMM matches the one stored in its non-volatile memory. If there is a mismatch, the absolute position will be marked as invalid (see AXIS#.SAFEPARAM.ABSOLUTEPOSVALID) and a safe homing will be required before using function that requires an absolute position (as SLP [Safe Limited Position]).


  • This parameter must be enabled for any functions that needs an absolute position.

Value

Description

0

Disabled

1

Enabled
