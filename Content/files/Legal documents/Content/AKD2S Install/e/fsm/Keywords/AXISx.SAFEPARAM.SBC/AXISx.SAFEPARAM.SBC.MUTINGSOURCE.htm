AXIS#.SAFEPARAM.SBC.MUTINGSOURCE

Description

This parameter allows the user to specify a safe input to bypass Safe Brake Control (SBC) and permit the brake to release regardless of whether SBC is active or not. The feature is referred to as ‘SBC Muting’ in documentation and .

Value Description

0

Not used

1

Safe input 1

2

Safe input 2

3

Safe input 3

4

Safe input 4

5

Safe dual input 1

6

Safe dual input 2

  • If using this feature, make sure it is safe to release the brake and there is not a heavy load being held solely by the brake.

Context

For complete SBC information refer to SBC (Safe Brake Control).
