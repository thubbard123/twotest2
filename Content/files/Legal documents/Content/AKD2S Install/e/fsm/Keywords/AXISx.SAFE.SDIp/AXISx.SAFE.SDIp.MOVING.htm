AXIS#.SAFE.SDIp.MOVING

Description

Indicates whether the axis is moving in the positive direction or not when Safe Direction positive (SDIp) is not active (when SDIp is active, if the axis cannot move in the positive direction, the fault reaction is activated).

Value Description

0

Not moving

1

Moving

Context

For complete SDI information refer to "SDI (Safe Direction)".
