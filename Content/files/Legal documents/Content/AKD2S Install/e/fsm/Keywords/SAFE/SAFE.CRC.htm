SAFE.CRC

Description

Returns the Cyclic Redundancy Check (CRC) of the complete safety parameters file stored in this device. The CRC returned is 0, if there are no safety parameters available.

Bytes 0..3: System CRC

SAFE.CRC is accessible through telnet.

CoE gives the CRC (Index: 0x37C3, SubIndex: 0).

Code

--> SAFE.CRC
0x31a564de

Context

For information on validation and CRC refer to ➜ Validation.
