AXIS#.SAFE.SS1_#.DEC

Description

Sets the deceleration rate for the corresponding safe stop instance. When the safe stop instance is executed via safe input or FSoE the drive will attempt to stop following the associated deceleration rate. If multiple safe stop instances occur at the same time, the instance with the fastest deceleration rate will be used.


  • If AXIS#.SAFE.SS1_#.MODE is set to fieldbus control, the drive will not decelerate on its own and this parameter is ignored.

  • If the deceleration rate is too slow for the configured maximum time (AXIS#.SAFEPARAM.SS1_#.TIMETOSTO), the drive will be disabled via STO. The maximum time, normal operating speed, and the deceleration rate should be configured to ensure the drive stops under control before the time expires.
