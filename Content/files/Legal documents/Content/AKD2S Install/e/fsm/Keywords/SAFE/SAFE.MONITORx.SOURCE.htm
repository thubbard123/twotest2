SAFE.MONITOR#.SOURCE

Description

Defines the source of the monitored safe variable SAFE.MONITOR#.DATA. Two variables can be configured at the same time using SAFE.MONITOR1.SOURCE and SAFE.MONITOR2.SOURCE. The source can be one of the following values:
