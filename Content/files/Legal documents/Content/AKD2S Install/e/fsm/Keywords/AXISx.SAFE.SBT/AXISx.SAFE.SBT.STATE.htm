AXIS#.SAFE.SBT.STATE

Description

Read the state of the Safe Brake Test (SBT) on the specified axis. The return value can be one of the following:

Value

Description

0

The test is not running.

1

The test is starting.

2

The test is started.

3

The brake is tested.

4

The test is stopping.

Code

--> AXIS1.SAFE.SBT.STATE
0

Context

For complete SBT information refer to SBT (Safe Brake Test).

Related safety parameters:

  • AXIS#.SAFE.SBT.ERROR
  • AXIS#.SAFE.SBT.MOVE
  • BRAKE#.SAFE.SBT.ELAPSED
