AXIS#.SAFE.SS2_#.INTERNALSTATE

Description

Indicates the internal state of the Safe Stop 2 (SS2) state machine.

Value

Description

0

Idle

1

Initial delay to initiate ramp down

2

Ramp down in progress

3

Target speed reached or SS2.MAXTIME delay time reached

4

Final state

5

Fault reaction

Context

For complete SS2 information refer to SS2 (Safe Stop 2).
