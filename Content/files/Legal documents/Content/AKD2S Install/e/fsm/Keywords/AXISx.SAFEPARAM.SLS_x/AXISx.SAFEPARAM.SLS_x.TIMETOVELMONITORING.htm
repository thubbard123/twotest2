AXIS#.SAFEPARAM.SLS_#.TIMETOVELMONITORING

Description

Indicates the maximum time before starting the velocity monitoring of the Safely Limited Speed (SLS) safety function. If the velocity is not below the limit when this time elapsed then the function is faulted and it activates the configured fault reaction.
