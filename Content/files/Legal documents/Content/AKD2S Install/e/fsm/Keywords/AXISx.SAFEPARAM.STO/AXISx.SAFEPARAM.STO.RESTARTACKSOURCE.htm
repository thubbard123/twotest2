AXIS#.SAFEPARAM.STO.RESTARTACKSOURCE

Description

Indicates the input to use to manually clear the Safe Torque Off (STO) once the STO input is cleared. Only used when the STO is configured for Manual Restart mode (see AXIS#.SAFEPARAM.STO.RESTARTACKMODE).

 

Value Description

0

Not Used

1

Safe Input 1

2

Safe Input 2

3

Safe Input 3

4

Safe Input 4

5

Safe dual input 1

6

Safe dual input 2
