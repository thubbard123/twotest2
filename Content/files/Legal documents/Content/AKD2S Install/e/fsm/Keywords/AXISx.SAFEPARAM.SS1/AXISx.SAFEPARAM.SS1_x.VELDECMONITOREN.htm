AXIS#.SAFEPARAM.SS1_#.VELDECMONITOREN

Description

Indicates if the velocity and deceleration monitoring is enabled for this Safe Stop 1 (SS1) instance. Velocity and deceleration monitoring can only be enabled if a safe encoder is configured in the safety parameter. When the velocity and deceleration monitoring is disabled (default value), SS1 is time based (SS1-t).

Disabled (Default)
