SAFE.DEVICECRC

Description

Returns the Cyclic Redundancy Check (CRC) of the device’s safety parameters dedicated to this device stored inside the complete safety parameters file. If there are no safety parameters available, the CRC returned is 0.

--> SAFE.DEVICECRC
0x72d5a4df

Context

For information on CRC and validation refer to ➜ Validation.
