AXIS#.SAFEPARAM.UNIT.ACCINUSER

Description

Indicates the acceleration increment in user units.

 

The safe system uses the following formula:

With:

  • accInEncUnit is AXIS#.SAFEPARAM.UNIT.ACCINENC
  • accInUserUnit is AXIS#.SAFEPARAM.UNIT.ACCINUSER
  • resInNbInc: the encoder resolution in counts
  • resInEncUnit: resolution in encoder units

Example

  • accInEncUnit is AXIS#.SAFEPARAM.UNIT.ACCINENC
  • accInUserUnit is AXIS#.SAFEPARAM.UNIT.ACCINUSER
Acceleration: accInUserUnit[accUserUnit] = accInEncUnit[encUnit/s^2]
Acceleration∶ 60[rpm/s]=1[rev/s^2]
Acceleration∶ 1[mm/s^2]=1000000 [nm/s^2]
