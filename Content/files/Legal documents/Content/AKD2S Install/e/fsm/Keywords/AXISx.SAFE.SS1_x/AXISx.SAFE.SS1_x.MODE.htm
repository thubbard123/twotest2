AXIS#.SAFE.SS1_#.MODE

Description

Sets the behavior of the drive when the corresponding Safe Stop 1 (SS1) input is triggered.

Value Description

0

Drive Controlled - Drive performs a controlled stop following AXIS#.SAFE.SS1_#.DEC deceleration rate.

1

Fieldbus Controlled - The drive will not perform any stopping action, the fieldbus master should stop the drive within the configured AXIS#.SS1_#.T amount of time before the drive triggers the STO signal. The manufacturer status object 50EFh 50EFh is used to signal to the fieldbus master that an SS1 input has been triggered.

Context

For complete SS1 information refer to SS1-t (Safe Stop 1)SS1-t/r (Safe Stop 1).
