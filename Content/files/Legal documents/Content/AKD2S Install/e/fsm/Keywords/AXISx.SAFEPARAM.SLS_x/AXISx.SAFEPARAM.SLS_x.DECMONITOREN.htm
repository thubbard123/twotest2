AXIS#.SAFEPARAM.SLS_#.DECMONITOREN

Description

Indicates if the deceleration monitoring is enabled for the Safely Limited Speed (SLS) safety function.

Value Description

0

Disabled (Default)

1

Enabled

 

Context

For complete SLS information refer to SLS (Safe Limited Speed).
