AXIS#.SAFEPARAM.STO.FUNCTIONACTIVATION

Description

Indicates how the function can be activated. For Safe Torque Off (STO), the function is ready for activation.

Value

Description

1

Ready for activation

 

Context

For complete STO information refer to STO (Safe Torque Off).
