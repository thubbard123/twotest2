AXIS#.SAFEPARAM.SBT.MAXAPPLOAD

Description

Specifies the maximum load of the application reflected to the motor shaft converted to current in mA. This value is scalar without regard to direction. It defines the 100% maximum application load. The Safe Brake Test (SBT) is done with 130% of maximum load (current).

 

Context

For complete SBT information refer to SBT (Safe Brake Test).
