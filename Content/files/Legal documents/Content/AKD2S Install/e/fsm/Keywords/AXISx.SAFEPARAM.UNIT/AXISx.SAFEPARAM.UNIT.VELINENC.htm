AXIS#.SAFEPARAM.UNIT.VELINENC

Description

Indicates the velocity increment in encoder units.

 

 

The safe system uses the following formula:

With:

  • velInEncUnit is AXIS#.SAFEPARAM.UNIT.VELINENC
  • velInUserUnit is AXIS#.SAFEPARAM.UNIT.VELINUSER
  • resInNbInc: the encoder resolution in counts
  • resInEncUnit: resolution in encoder units

Example

  • vellnEncUnit is AXIS#.SAFEPARAM.UNIT.VELINENC
  • vellnUserUnit is AXIS#.SAFEPARAM.UNIT.VELINUSER
Velocity : vellnUserUnit[velUserUnit] = vellnEncUnit[encUnit/s]
Velocity ∶ 60[rpm]=1[rev/s]
Velocity ∶ 1[mm/s]=1000000 [nm/s]
