SAFE.IDANDCRC

Description

Returns the Cyclic Redundancy Check (CRC) and the SafeID of the complete safety parameters file stored in this device. The CRC returned is 0, if there are no safety parameters available.

Bytes 0..3: System CRC
Byte 4: SafeID

SAFE.IDANDCRC is accessible through telnet.

Code

--> SAFE.INDANDCRC
0x1031a564de

Context

For information on validation and CRC, refer to "Validation" (➜ # 1).
