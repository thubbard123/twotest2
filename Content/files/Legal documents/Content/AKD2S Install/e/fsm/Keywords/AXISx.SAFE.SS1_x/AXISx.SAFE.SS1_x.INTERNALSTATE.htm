AXIS#.SAFE.SS1_#.INTERNALSTATE

Description

Indicates the internal state of the Safe Stop 1 (SS1) state machine.

Value Descrption

0

Idle

1

Initial delay to initiate ramp down

2

Ramp down in progress

3

Target speed reached or SS1.T delay time reached

4

Final state

5

Fault reaction

Context

For complete SS1 information refer to SS1-t (Safe Stop 1)SS1-t/r (Safe Stop 1).

Related safety parameters:

  • AXIS#.SAFE.SS1_#.DEC
