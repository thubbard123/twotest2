AXIS#.SAFEPARAM.SS1_#.TIMEFORVELZERO

Description

Indicates the time for the zero velocity. This can only be used when a safe encoder is configured in the safety parameters and speed/acceleration monitoring is enabled in the safety parameters (see AXIS#.SAFEPARAM.SS1_#.VELDECMONITOREN).
