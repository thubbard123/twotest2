AXIS#.SAFEPARAM.SDIp.FSOE

Description

Indicates if the Safe Direction positive (SDIp) safety function can be activated through FSoE. This parameter can only set to “Used” when FSoE is activated (SAFEPARAM.SAFEFIELDBUS.ACTIVATION is set to 1 – “Activated”). When activated, the safety system forbids the axis to move in the positive direction.

Value Description

0

Not used

1

Used
