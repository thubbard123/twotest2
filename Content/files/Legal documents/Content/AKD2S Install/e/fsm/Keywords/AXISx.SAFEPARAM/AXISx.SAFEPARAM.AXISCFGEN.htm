AXIS#.SAFEPARAM.AXISCFGEN

Description

Indicates whether the axis is enabled in the safety configuration. This parameter is for information only. It is handled automatically when using safety parameters from . When using FSoE parameters the defaults are properly filled out based on single or dual axis EtherCAT product codes.
