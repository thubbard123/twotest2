AXIS#.SAFE.STO.REPORTFAULT

Description

This parameter controls whether a Safe Torque Off (STO) fault F9000 will be created when STO is activated while the axis is enabled. The STO warning W9000 is always generated if this parameter is 1 and the STO function is active.

Value

Description

1

W9000 / F9000 are generated when axis is enabled and an STO occurs (default)

0

W9000 / F9000 are not generated

Context

For complete STO information refer to STO (Safe Torque Off).
