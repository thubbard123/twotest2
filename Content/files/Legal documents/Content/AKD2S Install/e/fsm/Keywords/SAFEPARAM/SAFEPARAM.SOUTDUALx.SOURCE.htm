SAFEPARAM.SOUTDUAL#.SOURCE

Description

When outputs are configured for dual outputs, this controls what safety function will trigger the safe output.


  • will not allow the user to select this setting unless the corresponding SOUTDUAL# is enabled. See SAFEPARAM.SOUTDUALx.EN.
