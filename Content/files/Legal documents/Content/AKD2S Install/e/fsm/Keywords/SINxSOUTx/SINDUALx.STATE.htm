SINDUAL#.STATE

Description

Reads the state of one safe dual digital input according to the number identified in the command. The safe dual input is composed of two physical safe inputs. The configuration of the safe dual input is determined by the safety parameters.

Usage

The state of the safe dual input is determined by the state of the two underlying safe inputs.

When dual channel mode is selected the drive supervises if the two inputs are in a different state for longer than 100 ms. If this occurs, an I/O failure fault is generated and the drive is put in safe-state.
