AXIS#.SAFE.SBT.SBA.ACTIVE

Description

Indicates the safe brake acknowledge state.

Value

Description

1

Brake needs an acknowledge.

0

Brake does not need an acknowledge. The brake state is safe.

Context

For complete SBC/SBT information refer to "SBC (Safe Brake Control)" (➜ # 1) / "SBT (Safe Brake Test)" (➜ # 1).
