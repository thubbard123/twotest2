AXIS#.SAFE.SSM_#.INTERNALSTATE

Description

Indicates the internal state of the Safe Speed Monitor (SSM) state machine.

Value Description

0

Velocity is not in range

1

Velocity is in range

Context

For complete SSM information refer to SSM (Safe Speed Monitor).
