SAFEPARAM.SIN.DISTINCTPULSE

Description

This setting indicates whether pulses on safe inputs must all be distinct from each other (i.e., offset from each other or different periods).


  • This does not disable the check for pulses on safe inputs. It only allows or disallows having the same pulse signal across safe inputs.
