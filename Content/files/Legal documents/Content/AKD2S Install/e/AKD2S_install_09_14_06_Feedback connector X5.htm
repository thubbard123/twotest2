Feedback Connector X5

Connector X5 is part of the single axis drives. Use this input for applications, which require a safe feedback.

 

  • 3 pin, pitch 3.81 mm
  • Second 2 wire feedback, SFA
  • Input for safe feedback.
  • Feedback types: SFD3, HIPERFACE DSL, EnDat 2.2
  • Required for SIL3 feedback control
  • Mating connector data (➜ # 1).
