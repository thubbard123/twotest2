EtherCAT® and Safety over EtherCAT®

drives (connectivity option E) can be connected as slaves to the EtherCAT® network (CoE) by using RJ45 connectors X11 (out port) and X12 (in port). The communication status is indicated by the built-in connector LEDs.

Single Flash = Local error
Double Flash = Process data watchdog

On/Blinking: Physical link/Data Traffic on.

„RUN“

Returns the device state:

Off = INIT
Blinking = PRE-OPERATIONAL
Single Flash = SAFE-OPERATIONAL
On = OPERATIONAL
Flickering = Initialization or BOOTSTRAP
