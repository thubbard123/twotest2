Functional Safety approval

offers 3 levels of functional safety implementation for :

  • Functional Safety Option 1: (➜ # 1), safe digital I/O command.
  • Functional Safety Option 2: (➜ # 1), safe digital I/O command or FSoE.
  • Functional Safety Option 3: (➜ # 1), safe digital I/O command or FSoE.

This manual is valid for drives with Functional Safety Option 123.
