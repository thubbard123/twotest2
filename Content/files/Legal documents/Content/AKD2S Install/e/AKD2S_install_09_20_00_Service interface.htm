Service Interface connector X20

  • RJ45 with built-in green/red dual-color LED
  • 100/10 Mbit EtherNet TCP/IP
  • Supports Auto-IP, DHCP and fixed IP addressing
  • Supports point-to-point (i.e. Auto-IP) and connection by using network switches
  • Supports automatic discovery in if in the same sub-net.

Operating, position control, and motion-block parameters can be set up by using the setup software on an ordinary commercial PC (➜ # 1).

Connect the service interface X20 of the drive to an EtherNet interface on the PC directly or by using a network switch, while the supply to the equipment is switched off. Use standard Cat. 5 EtherNet cables for connection (in some cases crossover cables will also work).

Confirm that the link LED on the (RJ45 connector) and on your PC (or network switch) are both illuminated. If both lights are green, then you have a good electrical connection.
