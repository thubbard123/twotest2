EtherNet/IP (available 2021)

with connectivity option I can be connected as slaves to a EtherNet/IP network by using RJ45 connectors X11 (Port 2) and X12 (Port 1). The communication status is indicated by the built-in connector LEDs.

 

Conn.

Name

Function

 

X11
Port 2

NS

Green - On = Connected
Green - Flashing = No connections

Red - On = Duplicate IP
Red Flashing = Connection Timeout

Red/Green Flashing = Self-test

Off = No power, No IP Address

X11
Port 2

Link/
Activity

On/Blinking: Physical link/Data Traffic on. Static off: No link.

X12
Port 1

MS

Green - On = Device Operational
Green - Flashing = Standby

Red - On = Major Fault
Red Flashing = Minor Fault

Red/Green Flashing = Self-test

Off = No power

 X12
Port 1

Link/
Activity

On/Blinking: Physical link/Data Traffic on. Static off: No link.
