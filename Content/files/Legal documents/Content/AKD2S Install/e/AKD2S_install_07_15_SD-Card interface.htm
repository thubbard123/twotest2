SD Card Slot

offer a SD card slot

  • for file transfers from/to the and SD Memory Card or
  • to boot the drive with data from the SD Memory card
  • or to set a fixed node ID or IP address.

These features can be started from the software or from the drive display using push-buttons B1/B2.
