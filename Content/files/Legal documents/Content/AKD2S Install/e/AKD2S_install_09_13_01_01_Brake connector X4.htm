Secondary Brake connector X4

Connector X4 is part of the single axis drives. Use for applications, which require a second safe brake for functional safety.

 

  • 3 pin, pitch 3.81 mm
  • Second (safe) motor brake
  • Required for SIL3 brake control
  • Mating connector data (➜ # 1).

 

X4 Pin

 Signal Description

1

PE

Protective earth

2

BR2-

Second (external) brake -

3
