Master-Slave control

Several can be connected as slave drives to another which acts as a master. The slave drives use the master's encoder output signal (EEO, (➜ # 1)) as command input and follow these commands (velocity and direction).

Master-Slave using connector X22

The master is configured for EEO2 (➜ # 1) on X22, the Slave uses X22 for command input.

Master-Slave using optional connector X23 or X41

The master is configured for EEO1 (➜ # 1) on X23 or EEO3/EEO4 ((➜ # 1) with SFA) on X41, the Slave uses X23 (or X41 with SFA) for command input.
