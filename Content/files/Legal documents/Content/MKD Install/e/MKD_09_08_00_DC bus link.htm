The DC bus connection between modules is realized by a slider X23.

The sliders are secured with one Torx (TX20 head) screw per pin.

  • All screws (on both modules) have to be tightened before switching on voltage: (➜ # 1).
  • Release all screws before moving the slider.
  • On the last (right end) -N Module in the system add the plastic cover and tighten the screws in order to ensure from accidental touch of energized parts.

DC Bus sharing -C-CA to Supercapacitor (SC)

-C-CA supports supercapacitor (SC) management.

Tested SC module specification

  • Five Maxwell Technologies* capacitor modules (BMOD0006 E 160 B02*) connected in series:
  • Maximum series DC voltage = 750 V
  • Rated capacitance per module = 5.8 F
  • Total calculated capacitance = 1.16 F
  • Maximum continuous current = 7 A
  • See manufacturer data sheet for all information
  • External fusing (FD1/FD2): (➜ # 1)
  • Wire cross section (➜ # 1), sufficient cross section is 6 mm².

 

*= Other SC types and other capacitance from other manufacturers are possible, contact support.

The objectives of DC bus sharing with an -C-CA device that is connected to a supercapacitor are:

  • Eco friendly (saves energy / lower power consumption)
  • Higher energy efficiency with DC bus sharing without braking resistor
  • Reduce operating cost (lower cost of operation)
  • Safe machine shutdown in case of mains failure
  • Increased productivity
  • Increased motor lifetime

The charging circuit for the connected supercapacitor is realized with current regulators. During the initial start-up of the system the charge circuit limits the inrush AC current and provides the energy charge to DC link, drives and supercapacitor. This functionality allows to utilize a smaller AC mains interconnect switchgear, fuses and wires for the machine.

The -C-CA charge circuit automatically charges the supercapacitor with a specified current based on the application condition during the normal operation. This action limits the input AC mains current and lowers peak power consumption. It also allows to the stabilization of the DC link voltage during the dynamic operation of multiple drives and recovers all kinetic energy during dynamic braking.

During short AC mains power outages this action provides a ride-through capability to the machinery that is operated by the drives and in case of an emergency the stored energy allows the active brake to standstill of the motion system.

The use of supercapacitors as energy storage element requires carefully managed charge/discharge cycles with the optimal level of applied voltage and current.

The supercapacitor must be discharged via an external regen resistor.
Example wiring for discharging (➜ # 1).

The manufacturer of the machine must generate a risk assessment for the sytem with supercap.

 

DC bus sharing examples

The following are the requirements for DC bus sharing with -C devices that are connected to a supercapacitor. For this purpose of DC bus sharing two -C devices can be operated in parallel which are each connected to multiple -N drives.

Two specifications can be proposed which use a supercapacitor to store or provide energy and in which the -C devices are connected in different setups and versions. The versions differ between an -C-CA version and -C-NA version.

Parallel connection of -C-CA and -C-NA

An -C-CA version and -C-NA version are connected in parallel to a supercapacitor. Only the MKD-C-CA version is connected to the AC mains and the -C-NA is only connected via DC bus to the -C-CA version and the supercapacitor.

Settings

Settings to be done for the -C-CA:

  • Check or set the current regulator value (max. 20A) via keyword VBUS.MAINSCURRENTLIMIT
  • Adjust Precharge timeout or set to 0s via keyword VBUS.PRECHARGETO
  • Save to device

Settings to be done for the -C-NA:

  • Set keyword VBUS.DCOPERATION to 1 for device DC operation
  • Save to device

The supercapacitor must be discharged via an external regen resistor.
Example wiring for discharging (➜ # 1).

 

Parallel connection of -C-CA and -C-CA

Two -C CA versions are connected in parallel to a supercapacitor. Both are connected to AC mains with separate 3L mains chokes and via DC bus to each other and the supercapacitor.

Settings

The following settings must be made for the every -C-CA:

  • Check or set the current regulator value (max. 20A) via keyword VBUS.MAINSCURRENTLIMIT
  • Adjust Precharge timeout or set to 0 s via keyword VBUS.PRECHARGETO
  • Save to Device

The supercapacitor must be discharged via an external regen resistor.
Example wiring for discharging (➜ # 1).

 

Discharging

 

High Voltage!

There is a danger of serious personal injury or death by electrical shock or electrical arcing.

  • The user must observe the safety information given by the manufacturer of the supercap modules.
  • The manufacturer of the machine must generate a risk assessment for the sytem with supercap.
  • Discharge process and discharge time must be analyzed.
  • Warning labels must be placed in the cabinet, if the discharge time exceeds 5 minutes.

Wiring Example

You must ensure, that discharging cannot be started as long as -C-CA is powered from AC mains.

The example below shows a 'Lock' contact which must be open as long as -C-CA is powered from AC mains. The contact should be controlled by the drive mains contactor.

Shut Down sequence

  • Safe Stop the application.
  • Switch of power supply for the -C-CA devices ('Lock' contact closes).
  • Close switch S0 (discharge) and keep it closed for the discharging time analyzed during the risk assessment.
  • Switch off mains power.
