When a module is ordered, the following items are included in the package:

-C package:

  • The ordered module itself
  • Printed copy of Product Safety Guide
  • DVD containing the setup software, , and all product documentation in digital format.
  • Mating connectors -C: X12, X13, X14, X15, X16, X25
  • One -N DC-Bus link cover
  • Two connector covers M23 for AKD-N

The M23 connector covers are required for protecting X2 of the last AKD-N in the strings.

-N package:

  • The ordered module itself
  • -N Data Sheet
  • Mating connectors -N: X9, X24, X26

Accessories:

Accessories must be ordered separately if required.

  • Mains choke (➜ # 1)
  • External regen resistor (➜ # 1)
  • Cables (➜ # 1)
  • Motor choke (➜ # 1)
  • Cabinet grommets (➜ # 1)

Accessories for AKD-N see regional Accessories Manual or AKD-N Installation Manual.
