Guide to setup

Presumptions for the setup:

  • The drive is properly mounted (➜ # 1).
  • Protective earth is connected (➜ # 1).
  • Mains supply voltage is connected (➜ # 1).
  • Auxiliary supply voltage is connected (➜ # 1).
  • String cables are connected (➜ # 1).
  • Safety Controller and PLC I/Os are connected (➜ # 1).
  • EtherCAT Master is connected (➜ # 1).
  • Service PC is connected (➜ # 1).

System addresses

To tune and configure the system, it is recommended to connect to the system.

The system can be accessed through two different physical communication channels:

  • Ethernet Service Port (X18) on the -C unit (➜ # 1)
  • EtherCAT Bus In port (X10) on the -C unit (➜ # 1)

Both communication channels allow a single point of connect to access the -C itself and all drives (-N’s, AKD-Ns) that are connected to this -C unit.

MAC Address

The unique -C MAC address is pre-defined by the manufacturer (see nameplate). Every -N or AKD-N drive axis has a unique MAC address as well.

IP Address

The -C service port X18 supports Auto-IP and Static IP addressing.

Auto-IP:
All network devices that are connected receive a unique IP address in the range 169.254.x.y.
To configure this, follow the next steps:

  • Set rotary switch S1 to "0" (➜ # 1). Power up the the -C.
  • By default, the network adapter of your windows PC that is used to connect to the -C should be configured for automatic IP assignment. If automatic assignment is not default, you need to reconfigure the TCT/IP adapter:
    Open Network & Internet settings -> Ethernet -> Change adapter options -> Right click on network adapter -> Select TCP/IP -> Properties -> tab General: set both entries to "automatically". Reboot the PC.
  • Connect PC network adapter with Ethernet cable point to point to -C service interface (RJ45 connector X18).
  • If the link LED on X18 is green/blinking, the connection is working.
  • Wait until the network adapter has finished the AUTO-IP assignment. This can be detected by checking the 7 segment display. Once the -C has received an new IP address it will be shown in the display.

Static IP:
The -C IP address can be set with the S1 rotary switch (➜ # 1).

looks for IP address in the subnet to detect devices in the LAN and start communication. All connected devices must have static IP addresses. Refer to the Online Help for more information.

EtherCAT Node Address

The EtherCAT node address is assigned automatically by the EtherCAT master. Refer to Online Help and EtherCAT manual for more information.

 

System Setup

Simple access to the system can be achieved by connecting a laptop or Service PC to the -C service port X18 with a standard Ethernet Patch cable. All drives that are connected to the -C can be accessed via X18.

The -C forwards the Ethernet packets to all connected drives and returns the response packets. Every connected axis has a unique MAC address and can be viewed as standalone Ethernet device (client).
