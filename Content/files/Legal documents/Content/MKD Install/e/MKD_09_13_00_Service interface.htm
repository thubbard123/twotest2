Service Interface (X18)

Operating, position control, and motion-block parameters can be set up by using the setup software on an ordinary commercial PC"Hardware requirements" (page 1).

Connect the service interface X18 to an Ethernet interface on the PC directly or via a network switch. The cable can be connected/disconnected to/from X18 with power on.

The X18 service port of an -C does not allow access to -N drives connected to another -C devices.

Confirm that the link LED on the -C (the green LED on the RJ45 connector) and on your PC (or network switch) are both illuminated. If both lights are illuminated, then you have a good electrical connection.
