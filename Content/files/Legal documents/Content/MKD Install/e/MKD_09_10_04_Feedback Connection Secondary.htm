Secondary Feedback Connection (X33)

It is possible to set up master/slave systems, use an external encoder as a command encoder, secondary feedback device (dual loop control) or connect the amplifier to a third party step controller. See the “Feedback 2” screen in for setup.

Don't use for primary motor feedback connection!

Connector X33 can be configured as an input for 5 V (TTL level) signals.

  • Sub-D high density 9 pin, male
  • Mating connector female
  • Input for secondary feedback
  • X33A for Axis 1, X33B for Axis 2 if applicable

X33B is available in dual-axis modules only.
