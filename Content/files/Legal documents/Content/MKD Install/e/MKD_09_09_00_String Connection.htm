Drive String Connection

-C support three strings to connect up to 42 drive axes: (➜ # 1). The strings are built of a DC-Bus and an internal fieldbus connection.

The DC-Bus is powered with 565 VDC to 680 VDC depending on the mains voltage in standard operation.

If the 3 phase mains voltage is not available, a 55 VDC stand-by power is available from the 24 VDC supply for emergency ramp down or brake situations.

 

 

Cable which exit the cabinet, should be threaded through grommets.

String 1 (central)

String 1 can be used for connecting central -N drive modules with up to 14 axes.

-N drives modules are connected via sliding terminals (X23, DC Power) and a short fieldbus cable (X22-X27, X28-X27, Fieldbus) to string 1. The string is limited to 30 kW rated power.

DC power (X23)

The DC power lines supply DC power for up to 14 -N drives in string 1. Connection for the DC power must be done by a sliding terminals (X23).

Strings 1 and 2 (central)

The number of central drive axescan be duplicated by using string 2 as an internal string.

  • -N drive modules in string 1 are connected via slider terminals (X23, DC Power) and a short fieldbus cable (X22-X27, X28-X27, Fieldbus) to string 1.
  • -N drive modules in string 2 are connected via sliding terminals (X23, DC Power) and a fieldbus adapter cable (X21A-X28, Fieldbus) . The combined strings 1 and 2 are limited to 30 kW rated power.

DC power (X23)

The DC power lines supply DC power for up to 28 -N drives in string 1 and string 2. Connection for the DC power must be done by a sliding terminals (X23).

Strings 2 and 3 (decentral)

Strings 2 and 3 can be used for connecting up to 28 decentral AKD-N drives. Refer to the AKD-N Installation Manual for technical description of these drives. Every string is limited to 10 kW rated power.

For connecting the first AKD-N to one of the DC strings the cable type MKC-HYB-z-AKN must be used. This cable is a Y type adapter with one hybrid connector on the AKD-N end and two connectors on the -C end with separate DC power and local fieldbus lines.

The hybrid cable AKN-HYB-z-AKN must be used to connect more AKD-N.

The hybrid cables are described in the appendix (➜ # 1).

DC power (X20A, X20B)

The DC power lines supply DC power for up to 14 AKD-N drives in one string. Connection for the DC power must be done by a mating connector with strain relief due to EMC reasons. This interface is connected via the MKC-HYB-z-AKN cable.
