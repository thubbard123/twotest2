Guide to Electrical Installation

-C

Remove the front protection cover:

Install the power supply module as follows:

  • Select cables according to IEC 60204 (➜ # 1) and the system topology (➜ # 1).
  • Install shielding and ground the modules (➜ # 1).
    Ground the mounting plate, the 24V supply GND and CNC-GND of the control system.
  • Install digital I/O lines
  • Install Safety I/O lines
  • Wire the device.
  • Wire the FAULT contact in series into the emergency off circuit of the system.
  • If required, connect the external regen resistor (with fusing).
  • Connect the -N drives. Close and secure all DC Bus sliders (X23).
  • Connect the required AKD-N drives to string 2 or string 3. For local AKD-N connections refer to the AKD-N Installation Manual.
  • Connect the digital inputs and the digital outputs, if required.
  • Connect the global STO inputs for the used strings (➜ # 1).
  • Connect the local STO inputs of the used drive axes (➜ # 1).
  • Connect the 24V power supply (required voltage value see electrical data (➜ # 1).
  • Connect the 24V STO power supply (PELV, required voltage values see (➜ # 1).
  • Connect the mains choke. Use shielded cable between choke and -C.
  • Connect the mains supply. Check maximum permitted voltage value (➜ # 1).
    Check proper use of residual-current circuit breakers (RCD, (➜ # 1))
  • Connect the PC (➜ # 1) for setting up the drive system.
  • Check the wiring and observe the system topology limits (➜ # 1).

Reinstall the front protection cover:

 

-N (string 1)

Install the drive module as follows:

  • Select cables (➜ # 1) in accordance with the planned system topology (➜ # 1).
  • Install grounding (PE) and ground the drive module (➜ # 1). Ground the mounting plates and the motor housings.
  • Install digital I/O lines
  • Install Safety I/O lines
  • Wire the drive:

  • String 1 Power: close the sliding terminal (X23) to the adjacent module. Verify the position is monitored as "closed". Fasten the terminal screws.
  • Cover the DC Bus contacts of the last -N with the DC-Bus link cover.
  • String 1 fieldbus: Connect the fieldbus ports of the modules (X22, X27, X28, (➜ # 1))
  • Dual cable motor connection:
    6 A and 12 A axes: Connect the motor power & brake to X29, max. length 40 m.
    24 A and 48 A axes: Connect the motor power to X29, max. length 40 m.
    24 A and 48 A axes: Connect the motor brake to X30, max. length 40 m.
    Connect the feedback to X32 , max. length 40 m.
  • Single cable motor connection:
    6 A and 12 A axes: Connect the hybrid motor power&brake to X29, max. length 40 m.
    24 A and 48 A axes: Connect the motor power to X29, max. length 40 m.
    24 A and 48 A axes: Connect the motor brake to X30, max. length 40 m.
    Connect the hybrid motor cable feedback to X31, max. length 40 m.
  • Check the wiring against the planned system topology (➜ # 1) and the wiring diagrams (➜ # 1).
  • Ensure the -N DC-Bus link cover is installed on the last -N before switching on mains power.

AKD-N (strings 2 or 3)

Install the drive as follows:

  • Select cables (➜ # 1) in accordance with the planned system topology (➜ # 1) and with AKD-N Installation Manual.
  • Install grounding (PE) and ground the drive, see AKD-N Installation Manual. Ground the mounting plates and the motor housings.
  • Wire the drive:
  • String 2: Connect the hybrid power cable between -C X20A/X21A and first AKD-N X1, max. length 40 m.
  • String 3: Connect the hybrid power cable between -C X20B/X21B and first AKD-N X1, max. length 40 m.
  • Connect the hybrid power cables between the AKD-N drives X1/X2.
  • Dual cable motor connection:
    Connect the motor cable to X4, max. length 5 m.
    Connect the feedback device to X5, max. length 5 m.
  • Single cable motor connection:
    Connect the hybrid motor cable to X4, max. length 5 m.
  • Check the wiring against the planned system topology (➜ # 1) and the wiring diagrams in the AKD-N Installation manual.
